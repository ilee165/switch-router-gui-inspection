## Goal

Add an "SSH" tab to `DeviceManagerDialog` showing stored host keys for the selected device, with a Delete button to remove individual keys. Uses `make_table()` from `panels/base.py` for consistency with all other tables in the app.

Not applicable — all operations in this plan run on the main GUI thread. `DeviceManagerDialog` is a modal dialog opened synchronously. `db.get_device_host_keys` and `db.delete_host_key` are called directly (no worker thread needed for DB reads of this size).

Adding the SSH tab requires wrapping the existing right-panel content into a `QTabWidget`:

### Task 2: Populate SSH Keys tab and implement Delete

**Type:** implementation
**File:** `device_manager.py`

**Add `_load_ssh_keys(self, device_id: int)` method:**

This method populates `self._ssh_table` with the host keys for the selected device. It is called when a device is selected in the list.

```python
def _load_ssh_keys(self, device_id: int):
    self._ssh_table.setRowCount(0)  # clear existing rows
    keys = db.get_device_host_keys(device_id=device_id)
    if not keys:
        self._ssh_empty_label.setVisible(True)
        self._ssh_table.setVisible(False)
        return
    self._ssh_empty_label.setVisible(False)
    self._ssh_table.setVisible(True)
    self._ssh_table.setRowCount(len(keys))
    for row_idx, key_row in enumerate(keys):
        set_cell(self._ssh_table, row_idx, 0, key_row["key_type"])
        set_cell(self._ssh_table, row_idx, 1, key_row["fingerprint"])
        set_cell(self._ssh_table, row_idx, 2, key_row["added_at"])
        # Store the key ID in the row for deletion
        item = self._ssh_table.item(row_idx, 0)
        item.setData(Qt.ItemDataRole.UserRole, key_row["id"])
```

The key ID is stored in column 0's item using `UserRole` data — same pattern as `list_widget` stores device IDs.

**Update `_on_item_clicked(self, item)`:** After loading the form, also load SSH keys:
```python
def _on_item_clicked(self, item):
    dev_id = item.data(Qt.ItemDataRole.UserRole)
    self._current_device_id = dev_id
    device = db.get_device(dev_id)
    if device:
        self.form.load_device(device, session_key=self._session_key)
        self._load_ssh_keys(dev_id)
```

**Add `_delete_host_key(self)` method:**

```python
def _delete_host_key(self):
    selected_rows = self._ssh_table.selectedItems()
    if not selected_rows:
        QMessageBox.information(self, "No Selection", "Select a key row to delete.")
        return
    # Get key ID from column 0 of the selected row
    row = self._ssh_table.currentRow()
    key_id_item = self._ssh_table.item(row, 0)
    if key_id_item is None:
        return
    key_id = key_id_item.data(Qt.ItemDataRole.UserRole)
    fingerprint = self._ssh_table.item(row, 1).text() if self._ssh_table.item(row, 1) else "?"
    reply = QMessageBox.question(
        self, "Confirm Delete",
        f"Delete stored host key?\nFingerprint: {fingerprint}",
        QMessageBox.StandardButton.Yes | QMessageBox.StandardButton.No
    )
    if reply == QMessageBox.StandardButton.Yes:
        try:
            db.delete_host_key(key_id=key_id)
        except Exception as e:
            QMessageBox.warning(self, "Delete Error", str(e))
            return
        if self._current_device_id is not None:
            self._load_ssh_keys(self._current_device_id)
```

Note: `db.delete_host_key(key_id=key_id)` uses the keyword-only argument defined in Plan 04-01.
