### Task 3: Cross-file keyword-only argument consistency check

**Type:** audit
**Files:** `db.py`, `host_key_dialog.py`, `device_manager.py`

Run these checks to verify keyword-only enforcement is consistent across all new functions:

```
python -c "
import inspect, db
fns = ['store_host_key','get_host_key','update_host_key','delete_host_key','get_device_host_keys']
for name in fns:
    fn = getattr(db, name)
    sig = inspect.signature(fn)
    bad = [p for p in sig.parameters.values()
           if p.kind in (p.POSITIONAL_OR_KEYWORD, p.POSITIONAL_ONLY)]
    status = 'PASS' if not bad else f'FAIL ({bad})'
    print(f'{name}: {status}')
"
```

Expected: 5 PASS lines.

```
python -c "
import inspect
from host_key_dialog import HostKeyVerifier
fn = HostKeyVerifier.verify_host_key
sig = inspect.signature(fn)
bad = [p for p in list(sig.parameters.values())[1:]  # skip self
       if p.kind in (p.POSITIONAL_OR_KEYWORD, p.POSITIONAL_ONLY)]
print('verify_host_key:', 'PASS' if not bad else f'FAIL ({bad})')
"
```

Expected: 1 PASS line.
