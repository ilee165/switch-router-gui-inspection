      Expected: Login succeeds normally. No error dialog. No slowdown visible

    Step 2 — Verify DB stores ciphertext (requires sqlite3 CLI or DB browser):
      Option A (command line):
        python -c "
import sqlite3
from pathlib import Path
conn = sqlite3.connect(str(Path.home() / '.switch_router_gui' / 'data.db'))
rows = conn.execute('SELECT name, password, enable_pass FROM devices').fetchall()
for r in rows:
    print(r[0], '|', r[1][:10] if r[1] else 'EMPTY', '...')
conn.close()
"
      Expected: Each device's password column starts with 'gAAAA' (Fernet token),
